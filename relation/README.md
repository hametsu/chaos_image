# Dynamic Relation Diagram #

## Requirements ##

* Sinatra
* Sequel
* Sqlite3
* Graphviz

## Makefile targets ##

* you can boot web system by "make web"
* "make target.png" makes graph from target.dot
* "make up" will upload graph image to gyazo.com
* default target dot file is defined in Makefile
